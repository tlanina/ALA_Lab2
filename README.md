## Що таке PCA?
PCA (Principal Component Analysis) — це метод, який:
- зменшує розмірність даних(робить дані простішими, але зберігає найважливішу інформацію)
  
- знаходить головні напрямки, у яких дані найбільше змінюються(головні компоненти)
  
У цьому завданні я використовую метод головних компонент (PCA) для стиснення зображення. Спочатку перетворюю кольорове зображення у чорно-біле, PCA стискає чорно-біле зображення, знаходячи “головні місця” (основні форми, перепади світла),
після чого застосовую PCA до матриці яскравостей. Визначаю мінімальну кількість компонент, що пояснюють 95% варіації. Далі відновлюю зображення з використанням цієї кількості компонент і порівнюю результат з оригіналом, а також візуалізую,
як змінюється якість від різної кількості компонент.
